Pi4j Demo

This is a simple demo I wrote to introduce ENSTA-Bretagne robotics students to the Raspberry Pi $35 computer.

It's a simple Pi4j (@Pi4j) that sets all the GPIOs of the Raspberry Pi pin header as outputs. You can then toggle each GPIO on/off, and drive DELs for example, using a simple graphical user interface.

You can find a pre-built jar in the dist/ folder. You need to run it as root.

$ cd Pi4jDemo/dist/
$ sudo java -jar Pi4jDemo.jar
